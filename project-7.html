Monitoramento Ambiental — Tecsus

Sistema de monitoramento ambiental para a Tecsus, com coleta, processamento e visualização de dados meteorológicos em tempo real. — 4° Semestre

GitHub

Sobre o Projeto

O projeto consiste no desenvolvimento de um sistema de monitoramento ambiental para a empresa Tecsus, permitindo a coleta, processamento e visualização de dados meteorológicos em tempo real. O sistema integra estações de medição físicas com uma plataforma web para exibição de dashboards e alertas ambientais.

A solução envolve hardware de sensoriamento conectado a uma interface web desenvolvida com React e TypeScript, armazenando os dados em MongoDB e MySQL para consulta histórica e análise em tempo real.

Minhas Contribuições

Fui responsável por toda a parte de hardware de medição — desenvolvimento, montagem e configuração das estações físicas de coleta de dados meteorológicos, incluindo a integração dos sensores e a transmissão dos dados para o sistema.
