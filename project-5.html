VISIONA GeoQuery

Plataforma web para análise de aspectos Ambientais, Sociais e de Governança (ASG) de propriedades rurais do Estado de São Paulo. — 6° Semestre

GitHub

Sobre o Projeto

O VISIONA GeoQuery é uma plataforma web desenvolvida para facilitar a análise de aspectos ASG (Ambiental, Social e de Governança) de propriedades rurais no Estado de São Paulo. A plataforma permite consultas geoespaciais em linguagem natural, gerando relatórios ambientais automaticamente com base em dados como terras indígenas, unidades de conservação, comunidades quilombolas e situação ambiental por município.

O sistema integra dados geoespaciais via PostGIS, visualização de mapas interativos com Leaflet e uma interface moderna construída em Next.js com Tailwind CSS. O backend é servido por uma API FastAPI em Python, conteinerizada com Docker.

Minhas Contribuições

Atuei principalmente no desenvolvimento backend, sendo responsável pela construção da API com FastAPI (Python), integração com o banco de dados geoespacial PostgreSQL/PostGIS, processamento de dados geográficos e configuração da infraestrutura com Docker. Contribuí também com partes do frontend em Next.js e TypeScript.
