Sobre o Projeto

Solução integrada que combina monitoramento ambiental em tempo real com controle de acesso. O sistema utiliza um microcontrolador ESP32 para coletar dados de sensores de temperatura, umidade (DHT11) e gás (MQ-9), enviando-os via WebSocket para um dashboard web interativo com gráficos em tempo real.

O módulo de controle de acesso usa um leitor RFID RC522 para identificar tags/cartões, registrando entradas e saídas no banco de dados MySQL. O sistema emite alertas automáticos via e-mail/SMS quando os limites ambientais são ultrapassados e suporta webhooks para integração com sistemas externos.

Minhas Contribuições

Desenvolvi o projeto inteiro — desde a programação do firmware do ESP32 em C++ (Arduino), montagem e configuração dos sensores e módulo RFID, até o servidor web com ESPAsyncWebServer, a comunicação via WebSocket e o dashboard HTML/CSS/JS com Chart.js para visualização dos dados.
