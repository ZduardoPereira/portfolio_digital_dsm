App Almoxarifado Militar

Aplicativo para gerenciar o almoxarifado militar da Brigada Fornovo di Taro, facilitando o controle de estoque e a logística de materiais. — 5° Semestre

GitHub

Sobre o Projeto

O projeto consiste no desenvolvimento de um aplicativo mobile e web para gerenciar o almoxarifado militar da Brigada Fornovo di Taro. O sistema facilita o controle de estoque de materiais, o cadastro de entradas e saídas, e a logística de distribuição de equipamentos, substituindo processos manuais por uma plataforma digital integrada.

A solução conta com um aplicativo mobile desenvolvido em Flutter, uma interface web administrativa em React, backend em Python e banco de dados PostgreSQL com Firebase para autenticação e notificações em tempo real.

Minhas Contribuições

Atuei principalmente no desenvolvimento backend, construindo as APIs em Python, integrando o banco de dados PostgreSQL e configurando o Firebase. Contribuí também com partes da interface web em React e TypeScript e com a aplicação mobile em Flutter.
